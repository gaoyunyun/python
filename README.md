Andy Hunt 介绍的主动阅读法：
- 调查（Survey）：扫描目录和每章总结，得出总体看法。
- 问题（Question）：记录所有问题。
- 阅读（Read）：阅读全部内容。
- 复述（Recite）：总结、做笔记，用自己的话来描述。
- 回顾（Review）：重读，扩展笔记，与人讨论。
还介绍了思维导图、以教带学等方法。
